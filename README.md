# Raja Rajeshwari Interior Works

Pan India Interior, Electrical, and Lighting Services. Founded and headed by Rajamouli Chary, bringing over 25 years of industry excellence.

## Project Structure

- `frontend/`: React + Vite + Tailwind CSS web application.
- `backend/`: Node.js + Express + MongoDB server API.

## Features
- Fully responsive mobile-app-like navigation interface.
- 3D Vision depth map photo rendering.
- Multi-lingual support (English & Telugu) with automatic admin translation tools.
- Click-to-call & WhatsApp connection buttons.
